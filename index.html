SimpleSlides 1.0

A single HTML document using CSS and jQuery to make a Powerpoint-like presentation

Made with love, by Jenn Schiffer.

Features

  • Your slides are a web page, so all slide content is HTML.
  • Completely customizable style-wise - just add to existing CSS.
  • Click navigation, as well as simple keyboard shortcuts (← / →)
  • Progress tracker - shows what slide out of how many you're on.
  • Cross-browser capability across the board - Chrome, Safari, Firefox, IE.
  • Looks great on the iPad, which is what I usually present these things on.
  • Well documented - read my inline comments and learn how it was made and how to edit.

Future changes

  • Rework scripts so providing slide ID #s is not required
  • Create a couple of simple external stylesheet themes for example
  • Not much else - "keep it simple, stupid!"
